# bpmn-vue3

#### 介绍
基于 Bpmn.js、Vue 3.x 、Element-plus 和vite的流程编辑器（前端部分），参考[bpmn-process-designer](https://gitee.com/MiyueSC/bpmn-process-designer)，支持监听器，扩展属性，表单等配置，可自由扩展

#### 软件架构
软件架构说明


#### 1. 安装依赖

```shell
npm install
// or yarn install
```

#### 2. 运行

```shell
npm run demo
// or yarn run demo
```
